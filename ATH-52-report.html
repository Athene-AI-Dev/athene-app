⚡ Features Implemented

💬

Thread Sidebar

Fetches threads from GET /api/threads. New Chat button creates threads via POST. Delete with confirmation dialog. Active thread highlighted.

🔄

Live Token Streaming

Uses fetch + ReadableStream to consume SSE from /api/agent. Tokens appended in real-time with loading spinner. AbortController for cleanup.

✋

Approval Cards (HITL)

Inline cards for pending email/calendar actions. Approve, Edit (JSON editor), and Reject buttons wired to POST /api/agent/approve.

📎

Citation Chips

[doc_id] markers rendered as hoverable chips with tooltips showing title, source type, and clickable external links.

⌨️

Composer

Auto-resizing textarea. Enter to send, Shift+Enter for newline. 10K char limit with visual counter. Attach/mic placeholder buttons.

🔗

Deep-linkable URLs

Thread ID persisted in URL via /chat/[threadId] routing. State reloads from /api/agent/status on mount.

📊

Analytical Mode

Toggle between Standard and Analytical modes via tab header. Analytical messages get "Business Intelligence Synthesis" label.

🛡️

useEventSource Hook

Reusable SSE hook with start/stop, AbortController, POST/GET support, and connection state tracking.
