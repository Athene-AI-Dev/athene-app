BUG-21

Recent Agent Decisions Panel is Empty With No Helpful CTA

Area Dashboard (Command Center)
Severity MEDIUM
Status Fixed

Issue Description

When the Command Center dashboard was loaded without any prior Human-in-the-Loop (HITL) agent decisions, the "Recent Agent Decisions" panel displayed a bare text message: "No agent decisions yet." It lacked any visual cues or call to action (CTA), taking up significant vertical space without providing guidance to the user. This made the dashboard feel incomplete and confusing for first-time users.

Root Cause

In app/(dashboard)/dashboard/page.tsx, the empty state mapping within the orchestrations.length === 0 condition only returned a div with large vertical padding (py-12) and simple text. There was no icon, no CTA button to direct users to the agent deployment area, and the padding was excessively large for an empty state.

Resolution

The empty state was overhauled to be more engaging and actionable. The vertical padding was reduced (py-8), a subtle visual illustration (an Activity icon) was added, the copy was refined, and an explicit CTA button ("Deploy Your First Agent") was introduced that redirects the user to the chat interface.

Code Implementation

// Before Fix
) : orchestrations.length === 0 ? (
  <div className="py-12 text-center bg-muted/5 rounded-[2rem] border border-dashed border-border">
    <p className="text-sm text-muted-foreground font-bold">No agent decisions yet.</p>
    <p className="text-xs text-muted-foreground/60 mt-2">HITL approvals and rejections will appear here.</p>
  </div>

// After Fix
) : orchestrations.length === 0 ? (
  <div className="py-8 text-center bg-muted/5 rounded-[2rem] border border-dashed border-border flex flex-col items-center justify-center">
    <div className="w-12 h-12 mb-3 rounded-full bg-primary/10 border border-primary/20 flex items-center justify-center">
      <Activity className="w-6 h-6 text-primary" />
    </div>
    <p className="text-sm text-foreground font-bold">No agent decisions yet.</p>
    <p className="text-xs text-muted-foreground mt-1 mb-4">HITL approvals and rejections will appear here once an agent runs.</p>
    <Button 
      onClick={() => router.push('/chat')} 
      className="h-9 px-5 rounded-xl text-[11px] font-bold uppercase tracking-widest transition-all shadow-sm"
    >
      Deploy Your First Agent
    </Button>
  </div>
)

Success: First-time users are now presented with a polished empty state containing an intuitive action button, improving the initial onboarding experience and optimizing dashboard layout.
